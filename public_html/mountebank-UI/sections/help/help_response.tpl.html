Help for Response Injection

Fields for response. See http://www.mbtest.org/docs/api/injection for more information on using requests and state.
Sample function
    function (request, state, logger) {
    logger.info('origin called');
    state.requests = state.requests || 0;
    state.requests += 1;
    return {
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({ count: state.requests })
    };
}
Field Type Default
statusCode int 200
headers object { "Connection": "close" }
body string ""
_mode string - binary or text text

Note that the Mountebank server must run with the --allowInjection flag. Mountebank Injection documentation
